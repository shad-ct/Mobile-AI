# Download Termux : 
```
https://play.google.com/store/apps/details?id=com.termux
```

# In termux : 
```
pkg update && pkg upgrade
pkg install git
pkg install nodejs
```


# Clone repository : 
```
git clone https://github.com/shad-ct/Mobile-AI
```

# Go to directory :
```
cd Mobile-AI
```
# find ip : 
```
ifconfig
```

# start server : 
```
node server.js
```

# Run Ollama : 
```
OLLAMA_HOST=0.0.0.0 ollama serve
```
